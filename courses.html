CUA Booking Prototype Se pueden mostrar los cursos como cartas o se pueden mostrar estilo lista. Cada clase puede tener su propia imagen, se deberian de implementar indicadores para poder separar mas facil la informacion tal vez el nombre del curson de pone de otro color. Cada curso tendria una lista de los temas discutidos en ellos. A los estudiantes se le puede hacer disponible una version de esta pagina la cual incluye los horarios de todos los mentores para asi determinar a que hora puede sacar cita antes de llamar. Otra manera de mostrar los cursos es teniendo una carta por categoria de clase de esta manera si el curso es de matematica se expanderian las opciones de la carta de Math y se buscan asi. Otra manera es el boton de filter para mostrar solamente los cursos de una materia.
